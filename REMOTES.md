# Git remotes

| Имя    | URL                                    |
|--------|----------------------------------------|
| origin | https://github.com/deathbringer777/ManicBot.git |
| gitlab | https://gitlab.com/deathbringer777/ManicBot.git  |

- **Текущая ветка:** `main`
- **Пушить на GitHub:** `git push origin main`
- **Пушить на GitLab:** `git push gitlab main`

В Cursor: **Source Control** (иконка ветки слева) → клик по ветке `main` → "Publish Branch" или "Push" — при пуше можно выбрать remote (origin / gitlab).
