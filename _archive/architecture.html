📋 KV Storage — What Stays vs What Migrated to D1

→ D1 ap:{id} → appointments
→ D1 u:{chatId} → users
→ D1 master:{chatId} → masters
→ D1 cfg:* → tenant_config
→ D1 blocked:{cid} → blocked_users
→ D1 ticket:{cid} → local_tickets
→ D1 hr:{cid} → human_requests
KV st:{chatId} — user state (TTL 2h)
KV rl:{chatId} — rate limit (TTL 60s)
KV chat:{chatId} — AI history (TTL 1h)
KV lang:{chatId} — language pref
KV bottoken:{id} — encrypted token
KV stripe:evt:{id} — idempotency (7d)
KV tktlock:{id} — distributed lock
